## Agent skills

All skills from Matt Pocock's collection are available globally and locally.

### Available skills:

- **diagnose** - Disciplined diagnosis loop for hard bugs and performance regressions
- **grill-with-docs** - Code review with documentation updates
- **grill-me** - Personal code review assistant
- **improve-codebase-architecture** - Improve overall codebase architecture
- **tdd** - Test-driven development with red-green-refactor loop
- **to-issues** - Convert tasks and requirements to trackable issues
- **to-prd** - Generate product requirement documents
- **triage** - Triage incoming issues with structured workflow
- **zoom-out** - Get a high-level view of the project
- **caveman** - Simple productivity skill
- **write-a-skill** - Guide for writing new skills

### Project configuration:

Skills are installed locally at: `skills/`
Also available globally at: `~/.config/opencode/skills/`

Run `/setup-matt-pocock-skills` to configure issue tracker and labels.
