# SerpenatriuStore

Proyecto de la empresa SerpenatriuStore.

## Estructura

- `docs/` - Documentación del proyecto
- `src/` - Código fuente
- `skills/` - Skills de desarrollo (Matt Pocock)
- `config/` - Archivos de configuración
- `scripts/` - Scripts de utilidad

## Skills Disponibles

Las skills de Matt Pocock están configuradas globalmente en:
- `~/.config/opencode/skills/`
- `~/.claude/skills/`

### Skills instaladas:
- **diagnose** - Diagnóstico de bugs
- **grill-with-docs** - Revisión de código con docs
- **grill-me** - Revisión personal
- **improve-codebase-architecture** - Mejorar arquitectura
- **tdd** - Desarrollo guiado por pruebas
- **to-issues** - Convertir tareas a issues
- **to-prd** - Documentos de requisitos
- **triage** - Clasificación de issues
- **zoom-out** - Visión general del proyecto
- **caveman** - Productividad
- **write-a-skill** - Crear skills

## Configuración

Ejecutar `/setup-matt-pocock-skills` para configurar el proyecto.
